# rust_refactoring_practice

Учебный проект по рефакторингу Rust-кода: парсер логов и CLI для чтения файла с логами.

Программа:
- читает лог построчно;
- парсит каждую строку в типизированные структуры;
- фильтрует записи по `request_id` и режиму чтения;
- умеет запускаться как библиотека и как бинарник.

## Что было улучшено

В проекте последовательно убраны основные антипаттерны из задания:

- парсеры переведены с `String` на `&str`, из-за чего исчезло много лишних копирований;
- дублирующиеся `just_parse_*` заменены на общий публичный API через трейт `Parse`;
- глобальный singleton-парсер убран, парсер строки лога теперь обычная зависимость;
- `Rc<RefCell<_>>`, trait object и `unsafe` в коде чтения логов заменены на дженерик `R: Read`;
- режимы чтения переведены с `u8`-констант на `enum ReadMode`;
- ручная фильтрация через циклы переписана на итераторы;
- проверка ненулевых чисел переведена на `NonZeroU32` и `NonZeroI32`;
- большой вариант enum уменьшен за счёт `Box<AuthData>`.

## Исправленные баги

Помимо рефакторинга были найдены и исправлены ошибки поведения:

- исправлена фильтрация в `read_log`: раньше при пустом списке `request_ids` фильтр по `ReadMode` фактически отключался из-за приоритета `||` и `&&`;
- добавлены тесты на фильтрацию для пустого и непустого `request_ids`;
- исправлен разбор `WithdrawCash`: раньше этот вариант журнала ошибочно создавался как `DepositCash`.
